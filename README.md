# Mission Control Dashboard

A clean, Linear-inspired dashboard for tracking tasks, projects, memory, docs and team. Built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- **Tasks**: Kanban board with Backlog/Todo/In Progress/Review/Done columns
- **Projects**: Progress cards with status, tags, and completion tracking
- **Calendar**: Monthly view with scheduled tasks and events
- **Memory**: Timeline of conversations and key decisions
- **Docs**: Searchable document library with category filters
- **Team**: Mission statement, active members, and planned roles

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Lucide React** for icons
- **GitHub Pages** for hosting

## Local Development

If you want to run it locally:

```bash
npm install
npm run dev
```

The dashboard will be available at `http://localhost:3001`.

## Usage

### Tasks
- **Backlog**: Ideas and future tasks
- **Todo**: Ready to start
- **In Progress**: Currently working on
- **Review**: Ready for approval
- **Done**: Completed tasks

### Projects
- **Active**: Currently working on
- **In Progress**: Making progress
- **Complete**: Finished projects
- **Planned**: Future projects

### Memory
- **Milestones**: Major achievements
- **Decisions**: Key choices made
- **Conversations**: Important discussions
- **Tasks**: Task-related context

### Docs
- **Search**: Find documents by title, summary, or tags
- **Categories**: Filter by type (Technical, Configuration, Reference, etc.)
- **Tags**: Additional metadata for filtering

### Team
- **Mission Statement**: Our purpose and goals
- **Active Members**: Current team members
- **Planned Roles**: Future team additions
